### How to install and run

Make sure you have node.js installed.
```
node -v
```

Clone into the repo.
```
git clone https://github.com/cekstedt/FreeCodeCamp-URL-Shortener.git
```

Navigate into the root project folder and install dependencies.
```
cd FreeCodeCamp-URL-Shortener/
npm install
```

Run the command `npm run dev` to start the development server, then navigate to http://localhost:3000/.
